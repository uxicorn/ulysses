Ads

Ads are everywhere and we have to account for them with our UI framework.

Ulysses' ad-blocks are always contained in an ad-wrapper and have a chained ad-unit class to the ad-block.

The ad-wrapper provides the optional lite-smoke background color and center alignment for inner text elements.

The ad-unit itself is a generic class that acts as a hook to show pointers on hover, while the actual dimensions are chained to the ad-unit with it's intended iab.net unit name (e.g. med-rect, billboard, leaderboard, etc.).

The "FPO" img element is for placement only and should be replaced with the ad script intended to fill the space.


Billboard

Billboard ads' height expand and collapse based on active state and often play interactive media. It's native size [970x90] is only opropriate for landscape oriented viewports 1024px wide and above.
